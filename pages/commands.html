Introducción

EdgaBot un Bot multifunciones para tu servidor, cuenta con comandos de moderación y automoderación para mantener seguro tu servidor.

Acerca del bot

Este Bot te ofrece multiples comandos y subcomandos de interacción, moderación, inteligencia artificial y minijuegos, esta alojado 24/7 y tiene actualizaciones constantes.

Tecnología

Este Bot esta hecha en varios idiomas y herramientas como: Discord.js, Replit, Autocode y Botghost.

Este Bot utiliza diferentes APIs que te hacen de tu servidor un lugar mejor y agradable.

Guía de inicio

Aprender a usar a EdgaBot es muy sencillo y practico, no necesitas saberes previos de algun tipo ya que el Bot te ayudara con aquellas dudas, si necesitas mas ayuda puedes unirte al servidor de soporte.

Ayuda del bot

Puedes usar el comando (e!help) para recibir información de algún comando.
No es muy dificil aprender las funcionalidades del bot, ya que cuenta con una buena descripción en cada comando a la hora de usarlo.

Importante: Debe estar de acuerdo con nuetra politica del Bot y no exdeder al uso de los comandos, pues pueden petar al Bot.
Use e!politica para mayor información.

Invitación al bot

Para agregar el bot a su servidor click aquí.

Esto necesita que usted sea el propietario del servidor o que tenga "Permisos de Administración".

Comandos

Syntax:
  • <> : Required Argument
  • [ ] : Optional Argument
  • ( ) : Command Aliases

Interacción

  • Anime
    • Descripción : Vamos a buscar cualquier anime con la ayuda de una API confiable.
    Usa : /search-anime < nombre de anime>
    Ejemplo : /search-anime < death note>
    Nota : El nombre de anime dado puede distinguir entre mayúsculas y minúsculas.
  • Memes
    • Descripción : Proporciono memes aleatorios de internet.
    Usa : e!meme
    Nota : Los memes actualmente son en ingles, pero se buscara una API en español.
  • Waifus
    • Descripción : Te proporciono imagenes de Waifus anime.
    Usa : e!waifus
    Nota : Por ahora este comando no contiene tags.
  • Acciones
    • Description : Realiza acciones con otros usuarios.
    Usa : /action < Tag> < @user>
    Ejemplo : /action < besar> < @Edgajuman>
    Nota : Por ahora existen 6 Tags: Saludar, Abrazar, Besar, Pegar, Gritar y Morder.
  • NSFW (Beta)
    • Descripción : Te proporciono imagenes de Waifus anime NSFW.
    Usa : /nsfw < tag>
    Nota : Por ahora este comando esta en versión BETA y solo funciona en los canales que el Owner registre en el Bot, puedes mandar el ID del canal en donde deseas que funcione el comando.
