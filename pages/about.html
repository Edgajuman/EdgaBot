Terminos de servicios

⊳ Al invitar a EdgaBot a tu servidor de Discord aceptas los términos y condiciones y nuestra política de privacidad.

⊳ Todos los servidores de Discord pueden usar a EdgaBot de forma gratuita y si alguien quiere cobrar por el servicio, comuniquese con nosotros en el servidor de soporte.

⊳ No puedes usar comandos del bot para dañarlo o intentar saturarlo.

⊳ EdgaBot contiene varias APIs que son utiles para mantener una buena convivencia entre los usuarios.

⊳ No puedes usar a EdgaBot para dañar otros servidores.

⊳ No puedes vender o comercializar a EdgaBot a través de otras plataformas como si fuera tuyo.

⊳ Mientras usas el bot, no puede usarlo de una manera que perjudique a otros usuarios con fines de troll.

Política de privacidad

》Para poder usar a EdgaBot correctamente debe otorgarle permisos de administración, ya que cuenta con multiples comandos de moderación.

》Cada usuario puede usar a EdgaBot como quiera, siempre y cuando no tenga la intención de dañarlo o dañar a otros.

》No puedes usar varios comandos al mismo tiempo simultaneamente.

》EdgaBot no guarda información personal de los usuarios, solo guarda el historial de chat + nombre de usuario o su ID.

》Si tu servidor no tiene acceso a algunos comandos puedes solicitarlo en el servidor de soporte del Bot.

》No hagas spam a la hora de usar la IA de EdgaBot porque puedes saturarla.

》Puedes usar subcomandos en MD para mayor privacidad de usuario.
